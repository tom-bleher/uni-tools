# uni-tools

A collection of tools to aid the Hebrew STEM student.

## Tools

### install_lyx (macOS)

One-click installer for [LyX](https://www.lyx.org/) with full Hebrew RTL + XeLaTeX support, based on the [Madlyx guide](https://mkali56.wixsite.com/madlyx).

```shell
chmod +x install_lyx/mac.sh
./install_lyx/mac.sh
```

Installs MacTeX, LyX, Culmus + Noto Hebrew fonts. Configures Hebrew RTL, David CLM for Hebrew / Latin Modern for English, F12 language toggle, and Cmd+E/Cmd+I for italic.

See [install_lyx/README.md](install_lyx/README.md) for full details, keybindings, and troubleshooting.
